On every report detail page, replace the current FULL REPORT PDF section with this layout:

1. A compact document access card.

Card design:
- dark / near-black panel
- thin green border
- subtle terminal styling
- no giant iframe by default
- no blank dead space
- no browser fallback message shown by default

Card content:
Micro label:
FULL REPORT PDF

Heading:
Official Archive Copy

Body copy:
This report is archived as a PDF document. Open it in a new tab for full-screen reading or download a copy for offline use.

- Report number
- Category
- Published date
- File type: PDF

- OPEN PDF
- DOWNLOAD PDF
- COPY PDF LINK
- PREVIEW INLINE

- OPEN PDF opens report.pdfUrl in a new tab using target="_blank" rel="noopener noreferrer"
- DOWNLOAD PDF links to report.pdfUrl and uses the download attribute
- COPY PDF LINK copies the full PDF URL to clipboard and shows a small “Copied” confirmation
- PREVIEW INLINE is collapsed by default. When clicked, it reveals the iframe below the card.

Important:
Do NOT show the iframe by default.
Do NOT show the giant blank PDF box by default.
Do NOT show “Your browser cannot display this PDF inline” unless the user opens Preview Inline and their browser fails.
